# Methodology

## Development process

We follow an iterative and incremental process based on short delivery cycles. Each iteration includes:

1. Joint planning with the client
2. Continuous development and testing
3. Review and validation
4. Deployment and feedback

## Code quality

Quality is non-negotiable. We systematically apply:

- **Code review**: All code goes through review before being merged.
- **Automated testing**: Unit, integration, and system tests.
- **Continuous integration**: Every change is automatically validated.
- **Living documentation**: Documentation is kept up to date as part of the process.

## Communication

We maintain open and regular communication channels with our clients. Weekly follow-up meetings, status reports, and direct access to the technical team.
